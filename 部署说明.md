# GalaxyLoom 部署说明

## 项目简介

GalaxyLoom 是一款基于 AI 技术的智能创作工具，提供完整的创作工作流支持。

## 系统要求

- **Node.js**: 23.11.1 或更高版本
- **操作系统**: Windows / Linux / macOS
- **内存**: 建议 2GB 以上
- **磁盘空间**: 建议 1GB 以上

## 快速开始

### 1. 安装依赖

```bash
npm install
# 或使用 yarn
yarn install
```

### 2. 启动开发服务

```bash
npm run dev
```

服务将在 `http://localhost:60000` 启动

### 3. 默认登录信息

- **账号**: admin
- **密码**: admin123

## 生产环境部署

### 方式一：使用 PM2（推荐）

1. 安装 PM2

```bash
npm install -g pm2
```

2. 构建项目

```bash
npm run build
```

3. 创建 PM2 配置文件 `pm2.json`

```json
{
  "name": "galaxyloom-app",
  "script": "build/app.js",
  "instances": "max",
  "exec_mode": "cluster",
  "env": {
    "NODE_ENV": "prod",
    "PORT": 60000,
    "OSSURL": "http://127.0.0.1:60000/"
  }
}
```

4. 启动服务

```bash
pm2 start pm2.json
pm2 startup
pm2 save
```

5. 常用命令

```bash
pm2 list              # 查看进程列表
pm2 logs galaxyloom-app # 查看日志
pm2 restart all       # 重启服务
pm2 stop all          # 停止服务
pm2 monit             # 监控面板
```

### 方式二：直接运行

```bash
npm run build
node build/app.js
```

## 环境变量配置

可以创建 `.env.local` 文件进行自定义配置：

```env
NODE_ENV=prod
PORT=60000
OSSURL=http://127.0.0.1:60000/
```

| 变量       | 说明                     | 默认值                      |
| ---------- | ------------------------ | --------------------------- |
| NODE_ENV   | 运行环境                 | dev                         |
| PORT       | 服务端口                 | 60000                       |
| OSSURL     | 静态资源访问地址         | http://127.0.0.1:60000/     |

## 常见问题

### 如何修改端口？

修改 `.env.local` 文件中的 `PORT` 变量，或在启动时指定：

```bash
PORT=8080 npm run dev
```

### 如何重置管理员密码？

删除 `db.sqlite` 文件，重新启动服务将自动初始化数据库，恢复默认账号密码。

### 服务无法启动？

1. 检查端口是否被占用
2. 检查 Node.js 版本是否符合要求
3. 尝试删除 `node_modules` 重新安装依赖
